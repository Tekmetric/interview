# Anime List

## Stack
- FE app:
  - React v19
  - TS v5
  - React DOM v19
  - RTK v2
  - MUI v7
  - styled-components v6
  - proxy-memoize v3

## Instructions
- Install the dependencies:
  - npm install
- In a separate terminal, start the proxy server:
  - cd server
  - npm run start
- Start the FE app:
  - npm run dev
