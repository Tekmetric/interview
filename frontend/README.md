# Tech Interview Project

## Steps to get started:

#### Fork the repository and clone it locally
- https://github.com/Tekmetric/interview.git

#### Let's install the project locally
`npm install`

#### Let's start the project locally
`npm start`

### Goals
1. Fetch Data from the backend Crud API you created or from a public API
2. Display data from API onto your page (Table, List, etc.)
3. Apply a styling solution of your choice to make your page look different (CSS, SASS, CSS-in-JS)
4. Have fun

### Submitting your coding exercise
Once you have finished the coding exercise please create a PR into Tekmetric/interview

## Miscellaneous
Curent solution uses NodeJs version 20.11.1 and the solution was developed with React using Typescript.

### Packages installed

- Material UI
- Material Design icons
- React Query
- Nanoid
- date-fns

### Public API used

For the current solution I have integrated the public API [Nager.Date](https://date.nager.at/)

1. The API ofers minimal information about a specified country and it's holidays. 
2. The list of available countries is also one of the features supported by the public API.
