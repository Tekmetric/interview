# Anime List

## Stack
- FE app:
  - React v19
  - TS v5
  - React DOM v19
  - RTK v2

## Instructions
- Install the dependencies:
  - npm install
- In a separate terminal, start the proxy server:
  - cd server
  - node src/proxyServer.js
- Start the FE app:
  - npm run dev
